Ход работы:
В первую очередь рассматривалась концепция игры. Есть 2 игрока, которые вытягивают карты из колоды до тех пор,пока у одного из игроков сумма очков не превысит 20. При 21 очке зачисляется победа, иначе проигрыш.
Игра разрабатывалась под одного ирока, следовательно за второго игрока действия выполняет алгоритм.
Алгоритм работы:
1. Выдается по одной карте каждому игроку.
2. Пользователю выдается еще одна карта и предлагается выбор взять еще 1 карту.
3. Пользователю предлагается еще карты, пока он не выберет "пасс", далее действует алгоритм бота.
4. Бот берет карты до тех пор, пока его сумма меньше 17, ход передается Пользователю.
5. Повтор 3 пункта.
6. Бот берет еще одну карту.
(На каждом этапе проверяется условие победы)

Перейдем к разбору кода:
Как колоду карт будем использовать массив, в котором будут указаны числа от 6 до 10, каждое повторяющееся 4 раза, также "J" "Q" "K" "A" по 4 раза. итого 36 карт.
Для перемешивания колоды будем использовать shuffle
Выдаются карты и рассчитываются их стоимости с помощью функции CardValueToInt:
Если выпало число, то в переменную записывается значение, иначе J = 2, Q = 3, K = 4
Если A, то 1, но если сумма всех очков меньше или равна 10, прибавляется 11.
PullOutACard выбирает случайную карту с колоды и удаляет ее из массива.
LoadProgress создает пустой файл, в котором будут сохраняться победы и поражения пользователя.
SaveProgress меняет значения побед и поражений на текущее.

Запуск программы осуществляется через ярлык.
